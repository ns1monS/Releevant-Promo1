Formularios HTML

A través de los formularios en HTML podremos enviar información al servidor que posteriormente éste procesará ya sea insertandola en una base de datos, enviando un email o cualquier otro proceso que el sistema requiera.

Etiqueta form

                    <form>
                    <!-- elementos del formulario -->
                    </form>

Atributos de un formulario

name nombre del formulario
action url a la que queremos enviar la información del formulario
method get | post (cómo se enviarán los datos)
target similar al de un enlace 
                
                    <form name="contacto" action="contactar.php" method="post" target="_blank" >
                    <!-- elementos del formulario -->
                    </form>

Elementos de un formulario

Inputs, select, textarea y botones

inputs

Todos tienen que tener un name que es nombre del parámetro y un id único (no obligatorio)

Text input

<input type="text" name="nombre">
Input Password

<input type="password" name="pass">
Input file

<input type="file" name="archivo">
Input hidden <input type="hidden" name="oculto">
Radio button input Otro

<input type="radio" name="genero" id="masculino" value="masculino" checked> <label for="masculino">Masculino</label> <input type="radio" name="genero" value="femenino"> Femenino <input type="radio" name="genero" value="otro"> Otro
Input Type Checkbox
Coche

<input type="checkbox" name="vehiculo1" value="bicicleta" checked> Bicicleta<br> <input type="checkbox" name="vehiculo2" value="Coche"> Coche

select

Menús desplegables



                    <select name="pais" id="pais">
                        <optgroup label="Europa">
                            <option value="ES" selected>España</option>
                            <option value="FR">Francia</option>
                        </optgroup>
                        <optgroup label="América">
                            <option value="AR" >Argentina</option>
                        </optgroup>
                    </select>
  • La etiqueta optgroup agrupa elementos dentro del desplegables (opcional)
  • La propiedad selected del option establece la opción seleccionada por defecto
  • La propiedad multiple que permite realizar selección múltiple

textarea

Atributos adicionales rows, cols y wrap

                    <textarea  name="observaciones" id="observaciones" rows="8" cols="50">Texto del textarea</textarea>

botones

Botón submit. Envia el formulario

<input type="submit" value="Enviar">
Input reset. Resetea el formulario

<input type="reset" value="Restablecer">
Botón button. Programación javascript

<input type="button" value="Contactar">
Etiqueta button

<button type"submit">Enviar</button>

Fieldset

Agrupación de campos

Datos personales 
                <fieldset>
                    <legend>Datos personales</legend>
                    <input type="text" name="nombre" placeholder="Nombre" style="margin-bottom: 8px;margin-top:10px"><br>
                    <input type="text" name="apellidos" placeholder="Apellidos">
                </fieldset>
