Listas HTML

Tipos de listas

  1. Listas no numeradas
  2. Listas numeradas
  3. Listas de deninifión

Listas no numeradas

  • Primer elemento
  • Segundo elemento
  • Tercer elemento
                    <ul>
                        <li>Primer elemento</li>
                        <li>Segundo elemento</li>
                        <li>Tercer elemento</li>
                    </ul>

Listas numeradas

  1. Primer elmento
  2. Segundo elemento
  3. Tercer elemento
                    <ol type="1">
                        <li>Primer elmento</li>
                        <li>Segundo elemento</li>
                        <li>Tercer elemento</li>
                    </ol>

El atributo type determina el tipo de orden: númerico o alfanúmerico

type="1" Numéricos (por defecto)
type="A" Alfabético en mayúsculas
type="a" Alfabético en minúsculas
type="I" Números romanos en mayúsculas
type="i" Números romanos en minúsculas
